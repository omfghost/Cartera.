EXPLORAR ACTIVO
Busca cualquier activo: acciones, ETFs, fondos, criptomonedas, índices o materias primas. Escribe su nombre (ej: "Apple", "Bitcoin") o su ticker (ej: AAPL, BTC-USD). Obtendrás precio en tiempo real, gráfico hasta 5 años y, en acciones, análisis fundamental con recomendación.
FUENTE DE DATOS · FMP▾
Para que Explorar y Analista funcionen de forma fiable, conecta una clave gratuita de Financial Modeling Prep (250 consultas/día, con datos fundamentales incluidos). La clave se guarda solo en este dispositivo.
DIAGNÓSTICO DE PRECIOS▾
Datos crudos guardados de cada activo. Si ves precios o monedas raros, pulsa "Resincronizar" para descargarlos limpios.

EDITAR ACTIVOS▾
Pulsa un activo para editar su nombre, tipo y ticker. Útil si la importación asignó mal el tipo o si necesitas corregir el ticker para que se sincronicen los precios.
IMPORTAR OPERACIONES DE BROKER▾
Importa tu historial desde un archivo CSV exportado de Trade Republic, Binance, DEGIRO, Interactive Brokers, XTB o MyInvestor. La app detecta el formato y añade las operaciones automáticamente.
Cómo exportar:
• Trade Republic: Perfil → Extractos → Exportar transacciones.
• Binance: Cartera → Historial → Exportar registros.
• DEGIRO: Actividad → Transacciones → Exportar CSV.
• Interactive Brokers: Informes → Flex/Actividad → CSV.
• XTB: Historial de cuenta → Exportar.
• MyInvestor: Movimientos → Descargar.
